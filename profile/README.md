# Throg

**Rust tooling for Web3 infrastructure** — SDKs, indexers, and Agent Skills for blockchain development.

## Projects

| Project | Description |
|---|---|
| [tronz](https://github.com/throgxyz/tronz) | Async-first Rust SDK for TRON, inspired by [alloy](https://github.com/alloy-rs/alloy). |
| [tronz-skills](https://github.com/throgxyz/tronz-skills) | Agent Skills (SKILL.md) that teach AI coding agents to build with tronz — providers, TRC20, contracts, staking, events. |
| [examples](https://github.com/throgxyz/examples) | End-to-end example projects built with tronz. |

markdown`tronz` is on [crates.io](https://crates.io/crates/tronz).
